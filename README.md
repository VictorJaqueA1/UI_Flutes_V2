# Proyecto Flautas

Base de trabajo para una plataforma que:

- permite visualizar y manipular dos flautas en una UI comparativa
- calcula curvas de inarmonicidad a partir de geometrías
- compara curvas mediante RMSE para emparejar flautas base e inversas
- persiste resultados operativos para la plataforma
- persiste resultados completos para replicación y trazabilidad técnica
