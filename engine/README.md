# engine

Aqui vive el motor de calculo de geometria, cortes, OpenWind y generacion de curvas.

Subareas principales:

- `models/`: estructuras del dominio de calculo
- `geometry/`: construccion geometrica para cada corte
- `constraints/`: validaciones del motor
- `cuts/`: protocolo de cortes
- `openwind/`: integracion con OpenWind
- `results/`: resultados limpios para backend y persistencia

Ya existe un caso minimo para una sola flauta y un solo corte.
