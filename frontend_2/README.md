# frontend_2

Interfaz activa del proyecto.

## Objetivo

Servir como plataforma visual para:

- visualizar flautas
- manipular sliders
- mostrar curvas
- lanzar cálculos
- preparar modos de exploración y cálculo masivo

## Importante

El estado visual actual de `frontend_2` se considera congelado y no debe alterarse accidentalmente al reorganizar arquitectura o documentación.
